# inception_v3_flowerIdentify
使用inceptionV3模型训练识别5种花朵

## 项目介绍
 
tensorflow框架下, 利用谷歌的inceptionV3模型训练自己采集的数据集，实现五种花朵（月季、绣球、万寿菊、三色堇、石榴花）的识别。
tonado搭建后端, h5+css+js完成前端页面。
